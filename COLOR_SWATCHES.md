## CSS Variables Quick Reference

### Main Brand
```css
/* Use these in your CSS */
hsl(var(--primary))              /* Bitcoin Orange */
hsl(var(--foreground))           /* Text color */
hsl(var(--background))           /* Background */
hsl(var(--secondary))            /* Dark Gray */
hsl(var(--muted))                /* Muted bg */
hsl(var(--muted-foreground))     /* Muted text */
hsl(var(--border))               /* Borders */
```

### The Corner
```css
/* Apply corner-theme class first, then use: */
hsl(var(--corner-background))    /* True Black */
hsl(var(--corner-primary))       /* Neon Orange */
hsl(var(--corner-accent))        /* Acid Green */
hsl(var(--corner-red))           /* Electric Red */
hsl(var(--corner-card))          /* Card bg */
```

## Color Combinations (Tested)

### Main Brand - Do's ‚úÖ
```
‚úÖ #f7931a on #ffffff (Bitcoin orange CTA on white)
‚úÖ #1a1a1a on #ffffff (Black text on white)
‚úÖ #ffffff on #1a1a1a (White text on dark)
‚úÖ #f7931a on #1a1a1a (Orange accent on dark)
‚úÖ #666666 on #ffffff (Muted text on white)
```

### Main Brand - Don'ts ‚ùå
```
‚ùå #666666 on #f5f5f5 (Low contrast)
‚ùå #f7931a on #f5f5f5 (Insufficient contrast)
‚ùå #e0e0e0 text on #ffffff (Invisible)
```

### The Corner - Do's ‚úÖ
```
‚úÖ #ffffff on #000000 (Maximum contrast)
‚úÖ #ff6b00 on #000000 (Neon orange glow)
‚úÖ #39ff14 on #000000 (Acid green accent)
‚úÖ #ff0000 on #000000 (Electric red warning)
‚úÖ #000000 on #ff6b00 (Black text on neon button)
```

### The Corner - Don'ts ‚ùå
```
‚ùå #ff6b00 on #ff0000 (Neon on neon - unreadable)
‚ùå Gray text on black (Breaks aesthetic)
‚ùå Muted colors (Against brand identity)
```
